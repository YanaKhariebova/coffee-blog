# ☕ Coffee Lovers Blog

A responsive blog for coffee lovers built with **HTML and CSS**.
The layout adapts to different screen sizes using **Media Queries**.

## 📌 Project Description

This project demonstrates a simple blog layout about coffee.
The design was recreated from a screenshot and implemented using **responsive web design**.

## ✨ Features

- Responsive layout (Desktop / Tablet / Mobile)
- Blog article layout
- Newsletter signup form
- Media Queries for different screen sizes

## 🛠️ Technologies

- HTML5
- CSS3
- Flexbox
- Media Queries
